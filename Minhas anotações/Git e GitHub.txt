Comandos gerais do Git
Mudar pastas - “CD/” colocar nome da pasta que quer e enter
Listas pastas - “DIR” 
Criar pastas/archivos - “MKDIR WORKSPACE” 
Deletar pastas/arquivos - “DEL” nome da pasta
Retroceder nível de navegação - “CD ..”
Limpeza de tela - “CLS”
Remover workspace - “RMDIR WORKSPACE”

SHA 1 -> algoritmo de encriptação -> encriptação gera conjunto de caracteres de 40 dígitos -> identificação de algoritmos de forma segura e rápida através dos dados encriptados 

Objetos do git 
Blob -> armazena um dado de uma forma mais específica -> metadados do git em formato encriptado do Sha
Tree -> armazenam blobs -> contém também metadados de blobs de Sha -> monta a estrutura dos arquivos blobs -> tem SHA das árvores também, encriptando as árvores em Sha também
Commit -> dá significado para árvores/blobs com o nome do autor e outros dados relacionados ao arquivo, como ele foi feito e quem o fez -> demonstra como o código está 100% seguro

Chaves SSh e Tokens
